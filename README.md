# DataForge: Enterprise Warehousing and Mining Platform

## Project Overview
DataForge is an enterprise warehousing and data mining project developed as part of the TCS iON Industry Project. The objective of the project is to transform raw business data into meaningful insights using ETL processes, database management and business intelligence tools.

The project demonstrates a complete workflow starting from data extraction and preprocessing to storage, analysis and dashboard visualization.

---

## Objectives

- Extract and preprocess sales data
- Implement ETL operations using Python
- Store processed data in MySQL
- Apply customer segmentation techniques
- Generate meaningful business insights
- Create an interactive Power BI dashboard
